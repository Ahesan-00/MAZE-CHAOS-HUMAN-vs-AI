MAZE-CHAOS-HUMAN-vs-AI
A Full-Stack Maze Competition Engine that facilitates a real-time challenge between a human player and an autonomous AI opponent.  
📱 Universal Compatibility
The engine is engineered with a mobile-first responsive approach, ensuring a seamless experience across all devices:  
Desktop: Optimized for high-resolution displays and keyboard input.  
Mobile & Tablets: Fully responsive layout with touch control support.  

🚀 Core Features
Dual Gameplay Modes:
Standard Mode: Play directly using the standalone JavaScript engine for a classic experience.
AI Challenge Mode: Connect to the Python backend to compete against an autonomous agent.
Hybrid Tech Stack: Features real-time synchronization between a JavaScript (ES6+) frontend and a Python (Flask) backend.  
Autonomous AI: The AI opponent utilizes advanced pathfinding algorithms, including A∗,Breadth-First Search (BFS),and Depth-First Search (DFS).  
High-Performance Rendering: Uses a responsive HTML5 Canvas rendering system for smooth, lag-free gameplay.  
Dynamic Audio: Includes a custom-built Web Audio API themed sound engine for an immersive gaming environment.  

🌐 Server Connectivity
To engage with the advanced AI engine, the application provides flexible connectivity options:
Localhost Access: Users can interface with the AI via localhost when the Python server is active on the host machine.  
Environment Setup: To enable AI features, ensure the Python server is installed and initialized to bridge the communication between the frontend interface and the AI logic.  
Standalone JavaScript: Alternatively, you can run the game in standard mode without a backend connection for immediate play.

🛠️ Technical Architecture
Frontend
Language: JavaScript (ES6+).  
Graphics: HTML5 Canvas API.  
Styling: CSS3 with adaptive layouts.  
Audio: Web Audio API.  
Backend
Framework: Python Flask.  
Logic: Implementation of pathfinding and maze generation algorithms.
